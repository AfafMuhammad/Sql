# eCommerce SQL Analysis Project

## Overview
This project demonstrates the use of SQL for analyzing an eCommerce dataset. The goal is to explore sales, customers, products, categories, and payments to extract actionable business insights.

## Dataset
The project includes six tables:
- **customers**: customer_id, customer_name, gender, state, signup_date
- **categories**: category_id, category_name
- **products**: product_id, product_name, category_id, price
- **orders**: order_id, customer_id, order_date, order_status
- **order_items**: order_item_id, order_id, product_id, quantity
- **payments**: payment_id, order_id, payment_method, payment_status
